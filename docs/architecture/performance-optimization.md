# 性能优化

## 前端优化

- **静态生成**: Astro 预渲染关键页面
- **代码分割**: 按路由分割 JavaScript
- **图片优化**: 自动格式转换和压缩
- **缓存策略**: 浏览器缓存 + CDN 缓存

## 数据库优化

- **索引设计**: 关键查询字段建立索引
- **查询优化**: 避免 N+1 查询问题
- **连接池**: 数据库连接复用
- **分页优化**: 游标分页替代偏移分页

## CDN 优化

- **全球分发**: Netlify CDN 全球节点
- **边缘缓存**: 静态资源边缘缓存
- **压缩传输**: Gzip/Brotli 压缩
- **HTTP/2**: 多路复用连接
